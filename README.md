# 🛒 Retail Sales Analytics: A Data Mining & BI Case Study using Tableau

## 📌 About the Project
A comprehensive Business Intelligence case study on Retail Sales Analytics using Python and Tableau. The project follows the complete Data Mining lifecycle — from problem definition to BI decision derivation.

## 🛠️ Tools & Technologies
- Python (Google Colab)
- Tableau
- Scikit-learn
- MLxtend
- Pandas, NumPy

## 🤖 Algorithms Used
| Algorithm | Task | Result |
|-----------|------|--------|
| Decision Tree | Sales Profitability Classification | 94% Accuracy |
| K-Means Clustering | Customer Segmentation | 4 Customer Groups |
| Apriori | Market Basket Analysis | Product Bundle Rules |

- 4 distinct customer segments identified for targeted marketing
- Cross-selling opportunities found between office supplies
